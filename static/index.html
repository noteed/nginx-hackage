Reesd Hackage Mirror

Use it.

This is a static mirror of the official Hackage server. To use this mirror as a package repository, change your cabal-install configuration (usually at ~/.cabal/config) by replacing the remote-repo line with:

remote-repo: hackage.reesd.com:http://hackage.reesd.com/


Read more.

You can read more about this Hackage mirror on the original GitHub repository: github.com/noteed/nginx-hackage.
